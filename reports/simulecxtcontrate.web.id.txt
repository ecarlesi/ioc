## Kit 1 — `scam2baru` (Xfinity credential/CC harvester)

### Description
Multi-stage phishing kit impersonating Xfinity. Collects login credentials, billing data, and full credit-card details (including double-card trick). Uses the **Gobot.su** bot-filtering service. Exfiltrates data via PHP `mail()`.

---

|-------|------|
| `freeusers2121@hotmail.com` | Recipient of all harvested data (configured in `config.php`) |
| `batak@gobot.com` | Spoofed "From" address in all exfiltration emails |

### Behaviour / TTPs

- **Brand impersonated:** Xfinity (Comcast)
- **Attack flow:** Sign-in → Billing → Credit Card → Security verification
- **Double-card trick:** First card submission returns "declined"; victim enters a second card. Both cards are collected.
- **Exfiltration channel:** PHP `mail()` to `freeusers2121@hotmail.com`
- **Bot evasion:** Gobot.su API + User-Agent blocklist + local IP cache (`ipbot.txt`)
- **GeoIP fingerprinting:** ipapi.is / ip-api.com / ipinfo.io
- **Admin panel:** protected by GET param `kuy=babilah321`; credentials `babilah` / `holong23`

## Kit 2 — `shortkontil` (URL shortener / redirect gate)

### Description
Minimal PHP redirect gate. Acts as a cloaked intermediary: legitimate visitors are redirected to a configurable URL (stored in `url.txt`); bots are sent to `127.0.0.1`. Uses the same **Gobot.su** API key as `scam2baru`.

---

### Credentials / Secrets

| Type | Value | Source |
|------|-------|--------|
| Admin panel password (plaintext) | `sayangdia12` | `index.php` (hardcoded) |
| Gobot API key | `7b1d2a9ae6e279ef93a4a428db08f03c` | `config.json` |

> **Note:** The Gobot API key is **identical** across both kits, strongly suggesting a single threat actor.

### Behaviour / TTPs

- **Role:** Cloaked redirect gate / URL shortener for phishing campaigns
- **Bot evasion:** Same Gobot.su API + local `ipbot.txt` cache
- **Admin authentication:** Single hardcoded plaintext password `sayangdia12` via POST `login_password`

- **Subdomain blocking:** Deflects mail/cpanel/webmail subdomains to `127.0.0.1`
